# 分诊标签 [Triage Labels]

工程技能 [Engineering Skills] 使用五个标准分诊角色 [Canonical Triage Roles]。本文件把这些角色映射到 GitHub Issues 中实际使用的标签 [Labels]。

| 技能标签 [Skill Label] | 仓库标签 [Tracker Label] | 含义 [Meaning] |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 维护者需要评估 [Needs Evaluation] |
| `needs-info` | `needs-info` | 等待报告者补充信息 [Waiting for Info] |
| `ready-for-agent` | `ready-for-agent` | 需求已清晰，可交给代理实现 [Agent-ready] |
| `ready-for-human` | `ready-for-human` | 需要人工实现 [Human-ready] |
| `wontfix` | `wontfix` | 不处理 [Won't Fix] |

当技能提到某个分诊角色 [Triage Role] 时，使用上表对应的仓库标签 [Tracker Label]。
